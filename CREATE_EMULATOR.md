# Create Android Emulator - Quick Steps

## In Android Studio (now opening):

1. Click **"More Actions"** (3 dots menu)
2. Select **"Virtual Device Manager"**
3. Click **"Create Device"** button
4. Select **"Pixel 5"** → Click **"Next"**
5. Click **"Download"** next to **"Tiramisu"** (API Level 33)
6. Wait for download → Click **"Next"**
7. Click **"Finish"**
8. Click the **▶️ Play button** to start emulator

## Then run in PowerShell:

```powershell
npm run android
```

Your app will install on the emulator!
